# 2557 Outreach Bot Code

**This is the code for the 2026 bot "Orpheus."**

To use this code, use:
```bash
# in your terminal
git clone https://github.com/GenshinAddict416/Outreach.git
```
This will create a new project folder called *Outreach* wherever you ran the command.

<br>
<br

If you are NOT team 2557, open **WPILIB VSC** and use <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> to open the command palatte.

Then, type <kbd>WPILIB: Set Team Number</kbd> and type in your team number.

<br><br>

Once your workspace is all configured, connect to the robot and run:
```text
./gradlew build; ./gradlew deploy
```

<br><br>

You're all done!
